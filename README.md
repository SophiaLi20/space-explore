#  Space Explorer

Space Explorer is an interactive web application that lets users explore the Solar System, stunning nebulae, and distant galaxies with smooth animations and high-quality space imagery.

##  Features

###  Exploration Categories
- **Solar System** – Sun, Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune  
- **Nebulae** – Orion, Helix, Eagle, Carina  
- **Galaxies** – Andromeda, Milky Way, Whirlpool, Sombrero  

###  UI Highlights
- Smooth click-based navigation  
- High-resolution NASA / ESA images  
- Glass-morphism panels  
- Modern dark theme  
- Responsive layout  

##  Technology Stack
- **HTML5**
- **CSS3 (Glass morphism, animations, gradients)**
- **JavaScript (ES6+)**

##  How to Use
1. Click a tab to switch categories  
2. Click the center image to browse objects  
3. View info on the left & features on the right
